# Auxiliador de Preços de Combustíveis

Aplicativo Android desenvolvido em Java que auxilia o usuário a decidir entre abastecer com álcool ou gasolina, com base nos preços informados.

O app utiliza a regra de que, se a razão entre o preço do álcool e o preço da gasolina for menor ou igual a 0.7 (70%), compensa abastecer com álcool. Caso contrário, a gasolina é a melhor opção.
